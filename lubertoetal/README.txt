Files associated with Luberto et al (2022).

Browndwarfs.txt is a file of all the brown dwarfs used, their astrometry, and our findings.
Nielson_check.ipynb is a notebook showing the results of running the code through Nielson and Bramich's findings.
Bd_microlensing_check.ipynb is a noetbook showing the creation of browndwarfs.txt

Note: The method for creating browndwarfs.txt, shown in bd_microlensing_check.ipynb is shown manually,
      but was done automatically with astropy's functional capabilities.
